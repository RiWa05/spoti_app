# Spoti-App

Spoti App aus Software Engeneering:
Idee: Eine App die anhand deiner Interessen Orte für dich Sucht die du besuchen kannst

Funktion: 
Veranstalter und Standorte können ihren Spot Hochladen 
Nutzer können ein Profil erstellen und Interessen angeben
Man kann alleine oder mit Freunden/ anderen Personen zsm suchen und alle vorhandenen Standorte werden mit euren Interessen verglichen und die besten Matches angezeigt 

Grobe Features:
Erstellen von einem Profil´(Nutzer oder Veranstalter) + Bearbeitung seines Profils
Hinzufügen von Standorten
Datenbank mit Standorten
Einladen von anderen Profilen in Gruppe
Such Funktion durch Filter(angebende Interessen)
Merken von Standorten + Kontakt(Mail) 
bewerten von Standorten (ohne Einfluss auf suche)
Anmelden und abmelden 
zeitliche terminierung

Optional: 
Auswahl für art des Treffens (date, alleine, mit freunden usw.)
Als Nutzer Orte zu erstellen
Premium version(Gruppen Funktion/ unendliche Suchanfragen pro Tag) + Bewerben von Standorten ohne Suche zu beeinflussen 
Ort Bewertungen von Nutzern fließen in Suche mit ein (anhand von tags/ stimme zu oder nicht)

App Ablauf:
Anmelden -> Profil erstellen -> Gruppe erstellen/ oder alleine -> suche Starten -> Abgleich Interessen mit Datenbank -> Ausgabe passender Standorte -> merken + Kontakt
